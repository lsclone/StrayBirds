####map

#####1. map<key, value>， key即first， value即second。

    map::iterator it = map.find(key);
    if(it != map.end())
    {
      it->first;  // key
      it->second; // value
    }

#####2. map[key] = value 解析： 不存在key，添加元素<key, value>；存在key，则赋值value给该元素的second

so, add new item, we can do it like this:

    if(map.end() == map.find(key))
    {
      map[key] = value;
    }
