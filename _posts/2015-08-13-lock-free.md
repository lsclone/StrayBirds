####2.、CAS(Compare and Swap)简要介绍

```
do{                             
  old_head = queue->head;        
  new_head->next = old_head;     
  if (old_head == queue->head){  
    queue->head = new_head;     
  }                              
}while(queue->head != new_head)
```
